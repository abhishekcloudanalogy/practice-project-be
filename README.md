# Backend Node/Express Template

Production-ready backend starter template using Node.js and Express.js.

## Features
- Express.js setup
- Modular folder structure
- Environment configuration
- Error handling middleware
- Health check route
- Logging with Morgan
- Security with Helmet
- CORS enabled

## Run Locally

```bash
npm install
npm run dev
```

## API

```bash
GET /api/health
```
